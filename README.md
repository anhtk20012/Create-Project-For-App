# How to run code created by create_folder.py
1. Open the project folder in VSCode and copy create_folder.py into it.
2. Open terminal in VSCode (Ctrl + `)
3. python create_folder.py will create all files and folders, create virtual environment (.venv) and install requirements.txt
4. To run the app, execute scripts run.bat

# Note: 
- Folder opened must be completely empty without any other files except create_folder.py
